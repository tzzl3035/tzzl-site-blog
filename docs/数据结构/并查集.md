### 简介

一种用于==管理元素所属集合==的数据结构。

### 大致思想

用一个森林，以==根元素==区分所属集合。

一般进行两种操作: ==合并==和==查询==。

### 例题

[【模板】并查集](https://www.luogu.com.cn/problem/P3367)

### 模板代码
```cpp{3-6}
int n;
std::vector<int> set(n);
auto find = [&](auto &find, int x) -> int {
  if(set[x] == x) return x;
  return find(find, set[x]);
};
```
