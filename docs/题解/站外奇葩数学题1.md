### 题目

解方程: $4^{-x} = x$

### 先不要看下面

:::tabs

@tab 提示
$0 < x < 1$

@tab 题解
### 解:
可以证明 $0<x<1$，此处舍去过程...\
$\frac{1}{4^x} = x$,\
$4^x = \frac{1}{x}$,\
${(4^x)}^{\frac{1}{x}} = ({\frac{1}{x}})^{\frac{1}{x}}$,*(huh?)*\
$({\frac{1}{x}})^{\frac{1}{x}} = 2^2$,\
$\frac{1}{x} = 2$,\
$x = \frac{1}{2}$

:::
